# Statbox

Statbox.py takes MongoDB connection count statistics and displays it on an 8x8 RGB LED matrix using a Raspberry Pi Zero. Full details of the project are in [The MongoDB Statbox](http://compose.io/articles/the-mongodb-statbox) article at [Compose](http://compose.io).

The code is available under an MIT license. Enjoy and please share any modifications you make so we can all enjoy a better Statbox.
